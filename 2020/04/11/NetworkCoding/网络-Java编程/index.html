3. 端口实现

3.1 概述

端口表示计算机上的一个程序的进程：

  • 不同的进程有不同的端口号！用来区分软件
  • 按规定分0——65535
  • TCP,UDP : 65535 * 2
    • TCP : 80
    • UDP: 80
    • TCP和UDP都可以占用80端口，但是在单个协议下，端口号不能冲突。
  • 端口分类：
    • 公有端口 （0-1023）
      • HTTP : 80
      • HTTPS : 443
      • FTP : 21
      • Telnet: 23
    • 程序注册端口：（1024–49151,分配用户或者程序）
      • Tomcat:8080
      • MySql:3306
      • Oracle:1521
    • 动态私有： （49152-65535）
