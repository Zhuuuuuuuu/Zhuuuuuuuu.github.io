null和undefined

  • null表示一个“空”的值，它和0以及空字符串’’不同
  • 0是一个数值，’’表示长度为0的字符串，而null表示“空”。
  • 在其他语言中，也有类似JavaScript的null的表示，例如Java也用null，Swift用nil，Python用None表示。
  • 但是，在JavaScript中，还有一个和null类似的undefined，它表示“未定义”。JavaScript的设计者希望用null表示一个空的值，而undefined表示值未定义。事实证明，这并没有什么卵用，区分两者的意义不大。大多数情况下，我们都应该用null。
