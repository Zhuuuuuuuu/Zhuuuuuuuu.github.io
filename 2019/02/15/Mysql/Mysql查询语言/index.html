DQL语言

DQL( Data Query Language 数据查询语言 )

  • 查询数据库数据 , 如SELECT语句

  • 简单的单表查询或多表的复杂查询和嵌套查询

  • 使用频率最高的语句

  • 是数据库语言中最核心,最重要的语句

2. 指定查询字段

  • 查询表结果时 , 可指定查询结果的数据列

    • 如 : SELECT * FROM student;

    • # 查询所有学生信息
SELECT * FROM student;

# 查询指定列(学号 , 姓名)
SELECT studentno,studentname FROM student;

4. 去重

  • 作用 : 去掉SELECT查询返回的记录结果中重复的记录(返回所有列的值都相同),只返回一条

5. 使用表达式的列

数据库中的表达式：文本值,列，NULL，计算表达式，变量

select 表达式 from 表

应用场景：

  • select 语句返回结果列中使用
  • select 与剧中的order by, having 等子句中使用
  • DML语句中where条件语句中使用表达式

6. where条件字句

  • 作用：检索数据中符合条件的值
  • 搜索条件可由一个或多个逻辑表达式组成 , 结果一般为真或假.

逻辑运算符

运算符 语法 描述
and && a and b 逻辑与
or || a or b 逻辑或
Not ! not a ! a 逻辑非

7. 模糊查询

比较运算符

运算符 语法 描述
Is NUll a is null 如果操作符是null ,结果为真
Is not NUll a is not null 如果操作符不为null,结果为真
Between a between b and c 如果a在b和c之间，结果为真
Like a like b SQL匹配，如果a 匹配到 b，则结果为真
in a in (a1,a2,a3) 如果a在a1,a2,a3中，结果为真
